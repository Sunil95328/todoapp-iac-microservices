This repository automates the creation and management of the cloud infrastructure for the ToDo microservices application.  
It uses **Terraform** to define infrastructure components in a reusable and version-controlled way.

The infrastructure includes:
- **Azure Kubernetes Service (AKS)** for containerized microservices.
- **Azure Virtual Network & Subnets** for secure networking.
- **Azure Container Registry (ACR)** for storing Docker images.
- **Azure SQL / Storage Accounts** for data persistence.
- **Network Security Groups (NSG)** for controlling inbound/outbound traffic.
